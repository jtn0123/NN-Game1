# Side-by-side eval: hand-crafted levels vs original CC1 reconstructions

Generated by `python -m experiments.cc_status.compare_originals` (pair_NN.png images
+ metrics table). Pairing is positional (my level N beside original level N) — the
hand-crafted levels are original designs, not remakes of specific caves.

## How much to trust the left side

The "original" images are the best-effort byte→tile reconstructions recovered from
git history. The decode is provably imperfect:

- **Credible structure (10/16):** CC1-01, 02, 04, 05, 07, 08, 11, 12, 13, 15 —
  solid mass in the plausible range (277–487 tiles); layout shape and density are
  meaningful.
- **Garbled (6/16):** CC1-03, 06, 09, 10, 14, 16 — terrain decoded as ladders/air
  (0–95 solids, e.g. CC1-06 shows "100 crystals" and 352 ladder tiles). Ignore
  these for structural comparison.
- **All object types dropped:** the decode maps levers, doors, enemies, hazards and
  pickups to blank/solid, so "orig has 0 enemies" in the metrics is a mapping
  artifact, NOT truth — research shows the real levels are full of them.

## What the images show (see pair_04 for the clearest credible example)

1. **Structure style — the biggest real gap.** The originals are *carved mazes*:
   thick terrain mass with winding 2–3-tile-tall corridors and irregular rooms.
   Mine are *open rooms with floating shelves* — regular tier spacing, lots of open
   air. Mine read as training gyms; the originals read as labyrinths.
2. **Collectible density.** Original caves are strewn with crystals (HUD 00/34 on
   CC1-04 vs 00/11 on mine), including clustered runs of 4–6. Mine place one
   crystal per shelf.
3. **Ladders confirmed.** Credible reconstructions show real ladder segments —
   settling the earlier "does the original have ladders?" question: yes.
4. **What already matches:** 40×24 scale, collect-all-crystals→exit rule,
   colour-keyed lever/door gates (all now genuinely gating), ladders, spikes,
   enemies, same engine art/HUD — and every level provably winnable, which the
   imported reconstructions never were (0/16).

## Recommended changes, in value order

1. **Raise crystal counts to ~25–40 per level** (biggest measurable authenticity
   gap; also lengthens episodes the way the real game plays).
2. **Add terrain mass on some levels**: convert open-air + floating-shelf areas
   into carved corridor mazes on 4–6 levels so the set spans both styles.
3. **More enemies** (4–8 per level) and broader hazard coverage.
4. Cosmetic: enable level dressing (signs/scenery) for the imported set.

Each change must re-pass: `analyze` (winnable), `analyze_gated` (lock ordering),
`door_value` (no decorative doors) — enforced by tests.
